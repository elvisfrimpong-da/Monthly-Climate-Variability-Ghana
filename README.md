# Monthly Climate Variability Ghana
Statistical and predictive analysis of monthly rainfall and temperature patterns in Ghana using satellite climate data, with applications to agricultural decision-making.
## Data Description

### Raw Data
The `raw_dataset/` file contains the original climate data downloaded from the NASA POWER database.
These files are preserved in their original form and were not modified.

### Processed Data
The `ghana_weather_final/` file contains cleaned and analysis-ready datasets derived from the raw data.
Data cleaning involved restructuring, validation, and preparation for statistical analysis.
